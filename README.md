# Mini Item Level Add-On (Modified Version)

## Disclaimer

This add-on is a modified version of the original [Mini Item Level](https://www.curseforge.com/wow/addons/mini-item-level) created by the original author. All rights belong to the original author.

## About this Modified Version

This version of the Mini Item Level add-on has been adapted to support:
- Korean language
- World of Warcraft Era
- Hardcore mode

No other changes have been made and all functionality should be consistent with the original add-on.

## Installation

1. Download the ZIP file for this modified add-on.
2. Extract the ZIP file into your `World of Warcraft/Interface/AddOns` directory.

## Usage

For usage instructions, please refer to the original [Mini Item Level documentation](https://www.curseforge.com/wow/addons/mini-item-level).
